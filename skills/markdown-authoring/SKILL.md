# markdown-authoring

## Instructions

- 見出しは ATX 形式の `#` を使い、下線形式は使わない。
- 見出しの前後には空行を入れる。
- 箇条書きの unordered marker は `-` を使い、`*` は避ける。
- 同じリンクを繰り返し使う場合は参照スタイルを優先する。
- 表は列幅を揃えて読みやすく保つ。
- Mermaid や mindmap のテーマは dark mode で見やすいものを使う。
